Content Advisor Integration Guide

Step 1 — Include the assets-selectors.js script, which exposes PureJSSelectors on the window.

<script src="https://experience.adobe.com/solutions/CQ-assets-selectors/static-assets/resources/assets-selectors.js"></script>

Step 2 — Call registerContentAdvisorAuthService() with your IMS auth props on page load.

// In order to obtain an imsClientId you will need to raise a support ticket with Adobe.
// Client Id's created via Adobe Developer Console will not work for Content Advisor.
const imsAuthProps = {
  imsClientId: IMS_CLIENT_ID,
  imsScope: "AdobeID,openid,additional_info.projectedProductContext,read_organizations",
  redirectUrl: window.location.href,
};
PureJSSelectors.registerContentAdvisorAuthService(imsAuthProps);

Step 3 — Call renderContentAdvisorWithAuthFlow() with a container element and props.

PureJSSelectors.renderContentAdvisorWithAuthFlow(container, {
  imsOrg: imsOrg,
  onClose: () => {},
  handleSelection: (assets) => console.log(assets),
});
